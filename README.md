# CareBridge HMS Prototype

CareBridge HMS is a hospital management system prototype for both single-doctor clinics and multi-speciality clinics.

## Included

- Patient, doctor, and hospital/admin role views
- Appointment booking
- OPD queue with queue number, room number, doctor cabin, and patient status
- Single-doctor and multi-speciality clinic modes
- Patient records and doctor note demo
- AI care assistant demo workflow
- Master prompt for building the production app with Google AI Studio and ChatGPT Codex

## Open The Prototype

Open `outputs/hospital-management-prototype.html` in a browser.

## Files

- `outputs/hospital-management-prototype.html` - working clickable HMS prototype
- `outputs/google-ai-studio-codex-hms-master-prompt.md` - production build prompt
